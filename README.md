# Portfolio React.js

Portfolio personale sviluppato con React.js, Tailwind CSS, Zustand e React Query.

## Tecnologie utilizzate

- React.js
- Vite
- Tailwind CSS
- Zustand
- React Query

## Installazione

```bash
npm install
```

## Avvio

```bash
npm run dev
```
